# 🌐 Core Web Vitals

Core Web Vitals are metrics defined by Google to measure **real-world user experience**.  
The three primary metrics are:

- **Largest Contentful Paint (LCP)** → Loading performance  
- **Interaction to Next Paint (INP)** / First Input Delay (FID) → Interactivity  
- **Cumulative Layout Shift (CLS)** → Visual stability 

## ⚖️ Summary Table

| Metric | What it Measures        | Good | Needs Improvement | Poor |
|--------|-------------------------|------|-------------------|------|
| **LCP** | Loading performance     | ≤ 2.5s | 2.5–4.0s | > 4.0s |
| **FID / INP** | Interactivity / responsiveness | ≤ 100ms (FID), ≤ 200ms (INP) | 100–300ms | > 300ms |
| **CLS** | Visual stability        | ≤ 0.1 | 0.1–0.25 | > 0.25 |
