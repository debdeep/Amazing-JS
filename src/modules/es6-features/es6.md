# 🚀 ES6 Features (2015)

- **let / const** → Block‑scoped variable declarations  
- **Arrow functions** → Concise function syntax, lexical `this`  
- **Classes** → Syntactic sugar for prototypes  
- **Template literals** → String interpolation with backticks `` ` ``
- **Default parameters** → Function parameters with default values  
- **Destructuring assignment** → Extract values from arrays/objects  
- **Spread operator (`...`)** → Unpack arrays/objects  
- **Rest operator (`...`)** → Pack remaining arguments into an array  
- **Modules (import/export)** → Native module system  
- **Promises** → Asynchronous programming pattern  
- **Iterators & Generators** → Custom iteration and lazy evaluation  
- **for...of loop** → Iteration over iterable objects  
- **Enhanced object literals** → Shorthand property
